# IOC Extractor (basic)
Extract IOCs (domains, IPs, emails, hashes) from text files, logs, or binary files (PCAP fallback).
- Output: JSON (summary + per-file) or CSV (flat).
- Safe for demos and IR exercises. Does not exfiltrate data.

## Quickstart

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python ioc_extractor.py -i sample_input.txt -o ioc_report.json
```

Files:
- `ioc_extractor.py`: main script
- `sample_input.txt`: example input with sample IOCs
- `ioc_report.json`: output generated by the demo
